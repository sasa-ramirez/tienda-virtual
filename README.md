# 🛒 Tienda Virtual — E-commerce Full Stack

Proyecto de aprendizaje y portafolio profesional: una tienda virtual completa construida con arquitectura escalable, pensada para crecer durante varios años.

## 🎯 Objetivo del proyecto

Aprender desarrollo Full Stack profesional mientras se construye una aplicación real de e-commerce, aplicando buenas prácticas de arquitectura de software (MVC, Repository Pattern, Service Layer, principios SOLID).

## 🚀 Tecnologías

**Frontend:** React, Vite, Tailwind CSS, React Router DOM, Axios

**Backend:** Node.js, Express, Prisma ORM, JWT, bcrypt, Cloudinary, Multer

**Base de datos:** PostgreSQL

## 📂 Estructura del proyecto

tienda-virtual/
├── frontend/     # Aplicación React
├── backend/      # API REST con Node.js + Express
├── database/     # Scripts SQL, diagramas ER
└── docs/         # Documentación técnica
